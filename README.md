# dipakwani.com

Personal site for Dipak Wani, built with Astro and deployed to GitHub Pages.

## Local development

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
npm run preview
```

## Content

- Projects live in `src/content/projects`
- Writing lives in `src/content/writing`
- Reading and listening notes live in `src/content/reading`
- Proof-of-work signals live in `src/content/signals`

The site preserves the public project routes:

- `/ai-mafia/`
- `/f1/`
- `/routine/`

Old Jekyll post URLs redirect to the migrated writing pages.
